# 🎬 YouTube Video Explorer

A modern and fully responsive **YouTube video explorer** built using React.
This application allows users to **search, browse, and watch videos**, along with advanced features like **live chat, debounced search, caching, and infinite scrolling** using the YouTube Data API.

## ✨ Features

* 🔍 Real-time video search with **debouncing**
* 💡 **Search suggestions (auto-recommendation)**
* 🎥 Watch YouTube videos seamlessly
* 💬 **Live chat functionality**
* ♾️ **Infinite scrolling** for continuous content loading
* ⚡ Optimized API calls using debouncing
* 🧠 **Redux for state management**
* 📦 **Caching search results in Redux store**
* 📱 Fully responsive design

---

## 🛠 Tech Stack

* React JS
* Redux (State Management & Caching)
* JavaScript (ES6+)
* HTML5

## 🔐 Environment Variables

Create a `.env` file in the root directory and add:

```bash
VITE_API_KEY=your_api_key_here
```

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/youtube-video-explorer.git
```

2. Navigate to project folder

```bash
cd youtube-video-explorer
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
